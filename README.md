# asdasd

This repository contains the `sdfdsfs` module - a simple Python implementation created to address the requirement.

## Files

- `sdfdsfs.py` - Main module with string manipulation utilities
- `test_sdfdsfs.py` - Test suite for the module
- `README.md` - This documentation file

## Usage

Run the module directly:
```bash
python3 sdfdsfs.py
```

Run the tests:
```bash
python3 test_sdfdsfs.py
```

## Functions

The `sdfdsfs` module provides:

- `greet(name)` - Generate greeting messages
- `reverse_string(text)` - Reverse any string
- `count_characters(text)` - Count character frequency in text

## Example

```python
import sdfdsfs

print(sdfdsfs.greet("World"))  # Hello, World! This is sdfdsfs.
print(sdfdsfs.reverse_string("hello"))  # olleh
print(sdfdsfs.count_characters("test"))  # {'t': 2, 'e': 1, 's': 1}
```
